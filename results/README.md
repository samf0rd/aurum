# results/

Each file is a JSON output from one backtest run.

Naming: `exp-NNN.json` where NNN matches the experiment YAML in experiments/.

Generated by:
  python run_experiment.py exp-NNN
  # or directly:
  python backtest_runner.py --profile intraday --from YYYY-MM-DD --to YYYY-MM-DD --out results/exp-NNN.json

Analysed with:
  python backtest_runner.py --analyse results/exp-NNN.json

```json
  "source": "backtest_runner",
  "generated": "<ISO timestamp>",
  "profile": "Intraday (M15)",

  "end_date": "YYYY-MM-DD",

  "stats": { "n_trades": 85, "win_rate": 0.294, "profit_factor": 0.47, ... },
  "trades": [
    {
      "entry_time": "...", "exit_time": "...",
      "entry_price": 1900.0, "exit_price": 1890.0,
      "direction": "LONG", "qty": 0.1,
      "pnl": -100.0, "r_multiple": -1.0,
      "exit_type": "stop_out",

      "regime": "TRENDING_BULL",
      "bars_held": 3
    }
  ],
  "equity_curve": [["<ISO timestamp>", 100000.0], ...]
}
```
